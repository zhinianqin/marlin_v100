# SM70 Dense NVFP4 Raw FP8 Scales Benchmark 对比

日期：2026-05-21

## 结论摘要

本次实验以 `8372ec27ef1e7fee21061c7697da686dc2e1a43f` 的 clean HEAD 作为 baseline，先完成 baseline build 与 SM70 dense nvfp4 全 15 个 CTA quick benchmark。随后将 dense nvfp4 的 `b_scales` 从预转换 fp16 语义改为 raw `torch.float8_e4m3fn` 语义，`global_scale` 改为 raw fp32 标量，并在 kernel metadata cache 阶段将 fp8 scale 解码、重排并缓存为 `half2`。

功能验证通过：

- `tests/test_marlin_helpers.py -k "nvfp4"`：`5 passed`
- `tests/test_marlin_dense.py -k "nvfp4 or cta_geometry or residue_n or small_tile or size_k"`：`100 passed, 100 deselected`

性能结论按 `kernel_like_us` 判断：

- 明确衰退：6 个 CTA
- 轻微退化：3 个 CTA
- 基本持平：1 个 CTA
- 改善：5 个 CTA

本次实验只记录事实；benchmark 退化未触发自动回滚，当前代码改动保持在工作区中，未自动提交。

## 代码改动摘要

- host 侧 nvfp4 判定改为 `b_type == float4_e2m1f && b_scales.dtype == torch.float8_e4m3fn`。
- nvfp4 必须提供 raw fp32 `global_scale`，且 `global_scale.numel() == 1`。
- `b_type == float4_e2m1f && b_scales.dtype == torch.float16 && global_scale` 不再走 nvfp4；mxfp4 仍保留 preconverted fp16 scales 且不使用 `global_scale`。
- `sm70_marlin_nvfp4_gemm.cu` 中 `b_scales` device pointer 改为 raw byte 语义。
- `cache_metadata_fp8_scales()` 读取连续 8 个 fp8 scale byte，调用 `dequant_fp8_scales<half2, vllm::kFE4M3fn.id()>`，再重排为旧 half2 cache 所需的连续 lane 对，并补回 FP8 exponent bias。
- nvfp4 FP4 qweight dequant 改为实际值路径，和 mxfp4 的 `vllm::kFE2M1f` 解码语义保持一致。
- Python helper 中 `marlin_quantize_nvfp4()` 返回 raw `torch.float8_e4m3fn` scales 和 raw fp32 `global_scale`；参考反量化改为 `fp4_values * fp8_scales.to(float32) * global_scale`。

## 观察

- raw fp8 scales 语义需要在 metadata cache 阶段做两件事：将 fp8 byte 解码为 fp16 数值，并将 `dequant_fp8_scales` 的 lane 配对重排为旧 half scale cache 使用的连续配对。
- 功能失败阶段曾暴露两个中间问题：缺少 FP8 exponent bias 补偿，以及 scale lane 配对和旧 half cache 不一致。最终修复后 targeted dense 测试全部通过。
- `128x256x8` 从 `81.32 TFLOPs` 降到 `73.55 TFLOPs`，是本次关注 CTA 中最明显的退化之一。
- `32x256x4` 是百分比退化最大的 CTA，`kernel_like_us` 变慢 `22.05%`。
- `256x64x4`、`64x256x8` 有明显改善，说明退化不是全局统一放慢，而是和 CTA shape 下的 cache/dequant 调度、寄存器生命周期、指令混排有关。
- 资源表显示无新增 spill；后续若要继续优化，建议优先看 SASS 中 fp8 scale cache 的 lane 重排与 FP4 dequant false 路径是否能重新折回更接近旧 preconverted half scales 的 codegen。
